# Explorando-factores-de-comportamiento-en-NovaRetail-
Análisis exploratorio de 15,000 clientes de NovaRetail+ para identificar los factores asociados con el ingreso anual. El proyecto utiliza Python, Pandas, visualizaciones y coeficientes de Pearson, Spearman, punto-biserial y V de Cramér para transformar patrones estadísticos en hallazgos de negocio
