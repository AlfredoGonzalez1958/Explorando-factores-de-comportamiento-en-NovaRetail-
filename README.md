Este proyecto analiza el comportamiento de 15,000 clientes de NovaRetail+ con el objetivo de identificar qué variables están asociadas con el ingreso anual generado por cada cliente.

El análisis incluye la revisión de la calidad de los datos, la clasificación de variables numéricas, binarias y categóricas, así como la exploración de relaciones mediante matrices de correlación, mapas de calor y gráficos de dispersión. Para obtener evidencia numérica se aplicaron los coeficientes de correlación de Pearson y Spearman, la correlación punto-biserial y la medida de asociación V de Cramér.

Los resultados muestran que el número de compras mensuales es la variable con mayor asociación con el ingreso anual. Las visitas a la plataforma y el gasto en publicidad dirigida también presentan relaciones positivas con la actividad comercial. En contraste, variables como edad, nivel de ingreso estimado, satisfacción, región y tipo de dispositivo muestran asociaciones débiles o prácticamente nulas.

El proyecto concluye con una interpretación orientada al negocio, la identificación de limitaciones y la propuesta de próximos pasos, entre ellos la segmentación conductual de clientes, el análisis del embudo de conversión, la evaluación de la eficiencia publicitaria y la aplicación de modelos multivariables y experimentos controlados.
